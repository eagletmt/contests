POJ 2665 - Trees

http://poj.org/problem?id=2665

概要

一本の長さ L の道に沿って，1m 間隔で木が植えられている． このうち，M 個の重複しない区間の木を切る． その後に残った木の本数を答える．

制約

  • 1 <= L <= 2000000000
  • 1 <= M <= 5000

解法

やるだけ．

 1 #include <iostream>
 2 using namespace std;
 3 
 4 int main(void)
 5 {
 6   int L, M;
 7   while (cin >> L >> M && L+M != 0) {
 8     L++;
 9     for (int i = 0; i < M; i++) {
10       int start, end;
11       cin >> start >> end;
12       L -= end - start + 1;
13     }
14     cout << L << endl;
15   }
16   return 0;
17 }
