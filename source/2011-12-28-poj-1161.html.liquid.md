## 概要
N 個の街があり，図のように M 個の領域(外側の領域も含む)に区切られている．

街 t から領域 r へ移動するのに横切る必要がある壁の数を f(t, r) で表わしたとき，
与えられた L 個の街 t[1], ..., t[L] について，f(t[1], r) + ... + f(t[L], r) の r に関する最小値を答える．

## 制約
- 3 <= N <= 250
- 2 <= M <= 200
- 1 <= L <= 30, L <= N

## 解法
領域同士の接続関係を表わすグラフをがんばって作ってワーシャル・フロイドした後，r に関する最小値を計算する．
